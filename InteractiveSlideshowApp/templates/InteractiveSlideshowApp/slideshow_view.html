This app is supposed to be a slideshow with the following properties:

  • Have a responsive design.
  • Controlled with 3 different types of input, including:
    • Mouse,
    • Keyboard, and
    • Speech

We're using Django web development framework (and Python) to be able to use Google Speech API for the speech control part. The rest of the project is HTML/CSS/JavaScript, using Bootstrap to make sure it's responsive.
